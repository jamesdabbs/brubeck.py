Welcome to Brubeck

Brubeck is a database of topological information, Ã  la the classic Counterexamples in Topology. It contains descriptions of several important topological spaces and properties and the interrelationships between each of them.

Features

Search by Traits

Since Brubeck stores information in a relational database, you can build complex queries to find spaces with a certain combination of traits. Not sure why a first-countable, separable space doesn't have to be second countable? Then search for an example of one that isn't.

Automated Proof Generation

Brubeck uses the theorems it has stored to generate proofs when possible. As soon as you tell it a space is metrizable, Brubeck knows that it is first countable, \(T_5\), \(T_4\) and so on.

Traceable Information

Every bit of data in the database should be traceable back to first principles. To facilitate this, Brubeck allows you to explore a proof's full implication tree.
