    Contributing to Brubeck

    Thanks for your interest in the Brubeck project. If you'd like to contribute, please try one of the following:

    • Add new Spaces, Properties, Traits or Implications
    • Add descriptions and proofs to items missing them.
    • Add proofs or counterexamples to the converses of known theorems.
    • Contribute code on GitHub

    If you have any other ideas, suggestions or bug reports, please don't hesitate to email me with them.
